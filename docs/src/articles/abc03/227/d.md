プロジェクト数 $x$ で割り当てができるかを考える.

$1$ つのプロジェクトには同じ部署の人がいてはいけないので, 各部署からは最大で $x$ 人出すことが可能である. これをプロジェクト $1$ の $1$ 人目, プロジェクト $2$ の $1$ 人目... と順に割り当てていき, $Kx$ 人を割り当てられれば可能である. すなわち, $\sum \min \{ A_i, x \} \geq Kx$ であれば可能である.

あとは $x$ を二分探索で求める.
